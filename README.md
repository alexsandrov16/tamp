# TAMP

Termux Apache MySQL PHP

# Instalación

Descargue o clone la última versión. '''git clone https://github.com/mayTermux/lampTermux'''
Mover al directorio
'''cd tamp'''
Ejecutar script de instalación
./tamp
## Información
URL Apache http://localhost:8080
Puerto predeterminado Apache 8080
Directorio $HOME/www
